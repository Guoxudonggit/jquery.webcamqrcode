WebcamQRCode

Scannez les codes QR à partir de votre site web

WebcamQRCode est un plugin jQuery qui utilise la webcam de l'utilisateur pour scanner un code QR et de retourner le résultat au Javascript afin d'être traité. Ce plugin a été développé à l'origine afin de scanner le code barre de produit et de remplir automatiquement les informations correspondantes sur le formulaire d'un intranet.

Ce plugin utilise Flash pour accéder à la WebCam.

ATTENTION: Ce plugin n'est plus maintenu car le code FLEX de LOGOSWARE sur lequel s'appui le plugin n'est pas compatible avec les version récente du player Flash.

>script>
	$('document').ready(function(){
		$('#qrcodebox').WebcamQRCode({
			onQRCodeDecode: function( p_data ){
					$('#qrcode_result').html( p_data );
			}
		});
		
		$('#btn_start').click(function(){
			$('#qrcodebox').WebcamQRCode().start();
		});
		
		$('#btn_stop').click(function(){
			$('#qrcodebox').WebcamQRCode().stop();
		});
	});
>/script>
